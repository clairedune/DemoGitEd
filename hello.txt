20 usefull git commands


Create a first repository
>> git init

Add all files to the staging zone
>> git add .

Status of repository
>> git status

Commit the files
>> git commit -m "remark"

Define your user-name
>>git config --global user.name <name>

Define your email
>>git config --global user.email <email>

initialize an empty directory (without working directory)
>>git init --bare 

Display states of physical repertories and staging zone. Non information on history.
>> git status

Display history of the project
>> git log

Display visual tree of the project
>> gitk

Commit
>> git commit -m "My commit name"

Change the name of the last commit
>> git commit --amend
